# Lilian Adegbola Projects

This repository contains the source code for Lilian Adegbola's web projects.

## Projects

### 1. Empowered Leadership Coaching Program Design
Located in: `empowered-leadership-coaching-program-design/`

This project focuses on the design and implementation of the Empowered Leadership Coaching Program website.

### 2. Membership Portal Design
Located in: `lilian-adegbola-membership-portal-design/`

This project contains the design and implementation for the Lilian Adegbola Membership Portal.

## Getting Started

To explore these projects, clone this repository:

```bash
git clone https://github.com/maxmarkagency/Lilian-adegbola.git
```

Navigate to the respective project directories to run them locally.
